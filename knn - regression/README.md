# Titanic KNN Regression (example)

A minimal KNN regression pipeline for the Titanic dataset. This mirrors the classification example but targets a numeric outcome using K-Nearest Neighbors regression.

## Project Structure

```
knn - regression/
├── data/
│   └── titanic.csv                 # Raw dataset
├── models/
│   └── knn_reg_model.pkl           # Trained KNN regressor (generated by notebook)
├── notebooks/
│   └── titanic_regression.ipynb    # Training notebook (placeholder)
├── app.py                          # Streamlit inference app for regression
├── requirements.txt                # Python dependencies
├── runtime.txt                     # Python version
└── README.md                       # This file
```

## Quick Start

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Place `titanic.csv` into `data/` (already copied).

3. Train the model by running the notebook in `notebooks/` and save artifacts into `models/`.

4. Run the app locally:

```bash
streamlit run app.py
```
